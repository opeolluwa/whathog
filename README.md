# Whathog
Backend of a Bilingual Mobile


## Getting Started
To get Started,
1. Clone the repository
2. Run `npm install`
3. create `.env` file and populate the fields appropriately
4. Run `npm run dev`  to run the project in development 


## Technologies
- [NodeJS](https://nodejs.org)
- [Express JS](https://expressjs.com)
- [TypeScript](https://www.typescriptlang.org)
- [TypeORM](https://typeorm.io)
- [PostgreSQL](https://www.postgresql.org)

## Extras
-  To run the project in production, run `npm run build` to build the sources
-  Obtain OpenAI API keys from <a href="https://beta.openai.com/account/api-keys">https://beta.openai.com/account/api-keys</a>
- Run `npm run db:migarte` to execute the database migrations
- Run `npm run db:revert` to rever the migration
